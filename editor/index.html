Add news or guide

Draft a post, then download the content file and copy the JSON entry to add to your repo. For guides, you can also edit an existing one.

Preview

After creating or editing a post:
1. Click Download .md file and save to content/news/ or content/guides/ (overwrite the file when editing).
2. Click Copy JSON entry, then open json/news.json or json/guides.json and paste (or replace the existing entry when editing).
