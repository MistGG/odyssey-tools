# SEO (Google / search)

The site is set up to be easily found on search engines.

## What’s in place

- **Meta description** on every page so search results show a short summary.
- **Open Graph and Twitter card** tags so links look good when shared (Discord, Twitter, etc.).
- **JSON-LD** `WebSite` schema on the home page for rich results.
- **robots.txt** so crawlers are allowed to index the site.
- **sitemap.xml** listing all main pages so Google can discover them.
